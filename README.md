Visualizing Self
===================

AUTHOR
---------
Jennifer Davis

DESCRIPTION
-------------

On June 20, 2013, Jennifer Davis gave a presentation on Visualizing Self at the Velocity 2013 conference. This repository is a place for 
the community to contribute helpful metrics and tools used to measure their work in an effort to come up with a common language to describe
work, measure work, and visualize work in our industry.

Project Principles
--------------------

Community is important and to be encouraged. 
 

Contributing
----------------

All contributions are welcome whether it comes in the form of ideas, patches, updates, emails, tweets, snail mail, or in person discussion.
